# Modern for EmulationStation Desktop Edition (modern-es-de)

The following options are included:

4 variants:

- Textlist with videos
- Textlist without videos
- Textlist with videos (legacy)
- Textlist without videos (legacy)

2 color schemes:

- Dark
- Light

4 aspect ratios:

- 16:9
- 16:10
- 4:3
- 21:9

3 transitions:

- Instant
- Instant and slide
- Instant and fade
