# Slate for ES-DE (slate-es-de)

The following options are included:

2 variants:

- Textlist with videos
- Textlist without videos

2 color schemes:

- Dark
- Light

4 aspect ratios:

- 16:9
- 16:9 vertical
- 4:3
- 4:3 vertical

3 transitions:

- Slide
- Instant
- Fade
